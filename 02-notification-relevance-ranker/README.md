# Notification Relevance Ranker

Goal: Build a small ranking demo that chooses which notification is most useful for a user.

This project will focus on simple relevance scoring. I want to test how user interests, freshness, urgency, and message type can change which item should be shown first.

## Planned version

The first version will use fake users and fake notification examples. It will not use real user data.

## What I want to practice

- Ranking items by score
- Comparing top results
- Testing simple relevance rules
- Writing down where the ranking works and where it fails
