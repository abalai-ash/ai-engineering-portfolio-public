# Portfolio Summary

This portfolio contains small, public-safe AI engineering projects built to show practical skills in applied machine learning, scientific computing, evaluation, reliability, and technical documentation.

The projects use synthetic data, toy examples, or public-safe examples. They are intentionally designed to be inspectable: each project explains the problem, implements a working prototype, includes evaluation logic, and documents limitations.
