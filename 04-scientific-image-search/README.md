# Scientific Image Search

Goal: Build a simple image search demo using public or fake image examples.

This project will focus on searching for visually similar images. I want to practice how images can be turned into features and compared in a simple way.

## Planned version

The first version will use public sample images or generated demo images. It will not use private research images or unpublished data.

## What I want to practice

- Loading image data
- Extracting simple image features
- Comparing images by similarity
- Testing whether the returned images make sense
