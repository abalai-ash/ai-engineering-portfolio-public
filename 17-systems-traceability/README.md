# 17. Systems Traceability

A compact requirements-traceability demonstration for a synthetic monitoring system.

The example connects stakeholder needs, requirements, component allocations, verification cases, and recorded outcomes. It identifies incomplete links and summarizes verification coverage.

## Included work

- requirement hierarchy
- parent-child traceability
- component allocation
- verification links
- missing-link checks
- coverage summary
- synthetic records only

## Run

```bash
python3 src/traceability_demo.py
```

## Test

```bash
python3 -m unittest discover -s tests -v
```

This public version is a limited demonstration. Broader review logic and evaluation cases are retained privately.
