# Agentic Research Workflow

Goal: Build a small workflow where an AI-style assistant uses simple tools instead of only giving a chat response.

This project will practice breaking a task into steps. The first version may include tools for searching notes, making a checklist, and drafting a short update.

## Planned version

The first version will use fake notes and simple local functions. It will not connect to private accounts or private files.

## What I want to practice

- Tool-based workflows
- Step-by-step task handling
- Human review before final output
- Keeping the workflow easy to inspect
