# 18. System Architecture Review

A compact architecture-review demonstration for a synthetic monitoring system.

The example connects system functions to logical components and physical resources, then checks interface and allocation completeness.

## Included work

- functional decomposition
- logical component mapping
- physical resource allocation
- interface records
- dependency checks
- concise architecture summary
- synthetic records only

## Run

```bash
python3 src/architecture_demo.py
```

## Test

```bash
python3 -m unittest discover -s tests -v
```

This public version demonstrates the main review behavior while broader analysis remains private.
