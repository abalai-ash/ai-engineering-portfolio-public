# Cloud ML Deployment

Goal: Build a small AI/ML demo with a project structure that could be deployed later.

This project will focus on making a simple app easier to run, share, and maintain. I want to practice separating code, data, settings, and notes.

## Planned version

The first version will be a small local app or API with clear run instructions. It will not include API keys or private data.

## What I want to practice

- Basic API structure
- Environment variables
- Logging
- Run instructions
- Deployment notes
