# 19. Verification Planning and Evidence

A compact verification-planning demonstration for a synthetic measurement system.

The example connects requirements to verification methods, planned cases, acceptance criteria, observations, anomalies, and closure status.

## Included work

- verification planning
- acceptance criteria
- evidence records
- anomaly tracking
- readiness checks
- requirement outcome summary
- synthetic records only

## Run

```bash
python3 src/verification_demo.py
```

## Test

```bash
python3 -m unittest discover -s tests -v
```

This public version contains a reduced dataset and a limited subset of the broader evaluation workflow.
